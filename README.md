# agrinho-2026
# objetivo
Este site tem como objetivo juntar tecnologia e sustentabilidade, querendo desenvolver uma fazenda inteligente, capaz de produzir e preservar recursos naturais ao mesmo tempo. Soluções como economia de água, cuidado com solo, proteção da biodiversidade e uso consciente da tecnologia. 
# justificativa 
Esta ideia surgiu apartir da visão de que na sociedade atual enfrentamos uma crise complexa, exemplo, usamos modelos de produção e consumo baseados na exploração de recursos com a necessidade de preservar o meio ambiente para as próximas gerações. Essa problemática global afeta o bem-estar dos ecossistemas e a qualidade de vida nas cidades.
